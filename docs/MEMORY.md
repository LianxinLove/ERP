## 记忆更新触发条件

更新以下文件时，同步更新本索引：

1. **新增阶段** - 在"项目阶段快速索引"中添加
2. **阶段完成** - 更新状态（⏳ → ⏺️ 或 ❌）
3. **新增文档** - 在"项目信息索引"中添加
4. **重大决策** - 更新最后更新时间

## 开发检查清单

每次开发开始前：
- [ ] 阅读 `CONSTRAINTS.md` 确认开发规范
- [ ] 阅读 `task_plan.md` 确认当前阶段
- [ ] 阅读 `progress.md` 确认上次进度
- [ ] 阅读 `findings.md` 确认技术决策

每次开发结束后：
- [ ] 更新 `progress.md` 记录今日工作
- [ ] 更新 `task_plan.md` 标记完成的任务
- [ ] 如有新决策，更新 `findings.md`
- [ ] 如有错误，记录到 `task_plan.md` 错误表格
