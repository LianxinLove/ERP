## 系统概述

ERP 系统是一个企业资源计划管理平台，旨在帮助企业管理业务流程、提高运营效率。

### 核心功能
- **用户认证** - 安全的用户登录和权限管理
- **组织管理** - 部门、职位、员工档案管理
- **审批流程** - 灵活的工作流引擎
- **采购管理** - 供应商、采购申请、采购订单
- **销售管理** - 客户、销售订单、销售退货
- **库存管理** - 仓库、产品、库存、调拨
- **财务管理** - 会计科目、应收应付、收付款

### 系统要求
- 现代浏览器（Chrome、Firefox、Safari、Edge 最新版本）
- 网络连接

## 快速入门

### 首次登录

1. 访问系统地址
2. 使用管理员提供的账号和密码登录
3. 首次登录后，建议修改初始密码

### 界面导航

系统采用左侧导航栏 + 顶部工具栏的布局：

- **侧边栏** - 主要功能模块入口
- **顶部栏** - 搜索、通知、用户菜单
- **主内容区** - 当前功能的操作界面

## 常见问题

### 登录问题

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码，或使用"忘记密码"功能（需配置邮件服务）。

**Q: 账号被锁定怎么办？**
A: 联系系统管理员解锁账号。

### 权限问题

**Q: 某些功能无法访问？**
A: 请确认您有相应的权限，如有需要请联系管理员分配。

**Q: 如何申请权限？**
A: 请向您的上级或系统管理员提交权限申请。

### 数据操作

**Q: 如何批量导入数据？**
A: 系统支持 Excel 导入功能，在相应模块中可找到导入按钮。

**Q: 删除的数据可以恢复吗？**
A: 大部分删除操作为软删除，可以联系管理员恢复。

### 审批流程

**Q: 待办任务过期了怎么办？**
A: 过期任务仍可处理，如有疑问请联系流程发起人。

**Q: 如何撤回已提交的申请？**
A: 在流程未进入下一节点前可以撤回。
